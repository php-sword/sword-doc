
## 运行环境

#### Linux操作系统
Swoole需要在Linux系统中运行，
你可以准备一台Linux服务器，或者通过虚拟机(具体方法百度)、Docker容器模拟Linux环境。

> 在 Windows 平台，也可使用 WSL(Windows Subsystem for Linux) 或 CygWin，这里不再单独说明
